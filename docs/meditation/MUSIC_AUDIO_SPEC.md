# 🎵 دليل الموسيقى والـ Ambient — واحة Meditation

> **المبدأ:** الموسيقى مش main character. هي **خلفية** تدعم التأمل.
> لو الـ user لاحظها بقوة = غلط. لو ما لاحظهاش = صح.

## 1. القرار الأكبر: من وين نجيب الـ Music؟

### 🥇 الخيار المُوصى به: **Artlist.io**
| العنصر | التفاصيل |
|--------|---------|
| السعر | **$200/سنة** (ممكن $120 مع خصم طالب) |
| المحتوى | 20,000+ track + sound effects |
| الترخيص | Royalty-free، commercial use، no copyright strikes |
| الجودة | 48kHz/24-bit WAV + MP3 |
| Arab/Egyptian vibe | فيه collection كامل "Arabic Middle East" |

**ليه مش غيره:**
- Epidemic Sound ($180/سنة): كويس بس أقل variety في middle-east
- YouTube Audio Library (مجاناً): جودة أقل + محدود

### 🥈 الخيار المجاني: **Chosic.com + Pixabay Music**
- مجاناً تماماً، CC-0
- جودة أقل، variety محدودة
- كافي لـ MVP

### 🥉 الخيار Premium: **تأليف مخصّص**
لو عايز حاجة unique:
- **Massive Grains** (مصري، SoundCloud): $300-500 للـ 5 tracks
- **Fiverr**: $50-200 للـ track، بس الجودة variable

## 3. Chimes / Bells (التنبيهات الصغيرة)

محتاجين **5 chimes قصيرة** — 2-4 ثواني لكل واحدة:

| الملف | الاستخدام |
|-------|-----------|
| `chime__session-start.mp3` | لما الجلسة تبدأ (بعد الـ 3-2-1) |
| `chime__session-end.mp3` | لما الجلسة تخلص |
| `chime__breath-in.mp3` | اختياري — tick خفيف لبداية الشهيق |
| `chime__breath-out.mp3` | اختياري — tick خفيف لبداية الزفير |
| `chime__milestone.mp3` | لما user يكمل journey (celebration) |

### Specs:
- المدة: **2-4 ثواني** + fade-out طبيعي (reverb tail)
- الأداة: **Tibetan singing bowl** أو **crystal bowl** (الأفضل)
- Volume: -14 LUFS (أعلى شوية من الـ ambient)

### من وين تجيبهم:
- Artlist > SFX > "meditation bell"
- Freesound.org (مجاناً، search "singing bowl")
- **Zapsplat.com** (مجاناً مع account)

**مرجع صوتي:**
- Headspace session end
- Insight Timer bell sounds

## 6. Loop Preparation (مهم للـ ambient)

الـ MP3 عادي ما بيلوبش perfectly (فيه سكتة صغيرة لما يرجع للبداية).

### الحل:
1. افتح الـ track في **Audacity**
2. استخدم **Crossfade Tracks** — 3 ثواني من آخر الـ track تعمل crossfade مع أوله
3. Export تاني

**Alternative:** نستخدم Web Audio API في الكود ونعمل gap-less playback automatically — ده اللي هاعمله.
