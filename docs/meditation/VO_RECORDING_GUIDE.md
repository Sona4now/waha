# 🎙️ دليل تسجيل الصوت — واحة Meditation

> **مهم:** اقرأ القسم ده كله قبل أي تسجيل.
> الدقة هنا بتحدد جودة التجربة كلها.

---

## 1. المعدات المطلوبة

### الحد الأدنى (كافي جداً)
- **ميكروفون**: Shure MV7 (~$250) أو Rode NT1-A مع audio interface
- **سماعة ستوديو**: أي closed-back headphones (Sony MDR-7506 أو ATH-M50x)
- **غرفة**: أصغر غرفة في البيت، مليانة سجاد/ستارة/سرير (ده بيمتص الـ echo)
- **Pop filter**: ضروري (بـ 50 جنيه من أمازون)

### البرنامج
- **Audacity** (مجاني) أو **Reaper** ($60) أو **GarageBand** (ماك فقط)
- Export: **MP3 128kbps stereo** + **WAV backup** (للأرشيف)

---

## 2. إعدادات التسجيل

| الإعداد | القيمة |
|---------|-------|
| Sample rate | **48000 Hz** |
| Bit depth | **24-bit** |
| Channels | Mono (VO بس) |
| Mic distance | **10-15 سم** من الفم |
| Gain | الـ peak بين **-12dB و -6dB** (ماتخليه يوصل صفر) |

### قبل ما تبدأ أي جلسة:
1. اشرب مي (مش قهوة) قبل التسجيل بـ 15 دقيقة
2. سخّن صوتك (تمارين صوت 5 دقايق)
3. اعمل **test recording 30 ثانية** واسمع التلات حاجات:
   - فيه echo؟ اتحرك لغرفة أصغر
   - فيه hiss في الخلفية؟ قفل المكيف/الثلاجة
   - صوت التنفس بين الجمل عالي؟ اتحرك شوية لشمال الميكروفون

---

## 3. طريقة الأداء — الأهم من المعدات

### القاعدة الذهبية
> **"اتكلم كأنك بتحكي لصاحب مُرهق قبل النوم — مش كأنك بتقرأ كتاب."**

### التفاصيل:
| العنصر | التعليمات |
|--------|-----------|
| **السرعة** | أبطأ بـ 30% من الكلام الطبيعي (حوالي 110 كلمة/دقيقة) |
| **النبرة** | دافية، حنونة، من الحنجرة الأسفل (chest voice) |
| **الصوت** | متوسط، **مش هامس** — هامس بيوتّر، دافي بيريّح |
| **الوقفات** | بعد كل جملة... **3 ثواني صمت** (مش ثانية) |
| **التنفس** | لو سمعت نفسك، اعمل edit وشيله. لكن كلّم طبيعي |
| **Signature phrase** | "أنا معاك" أو "خد وقتك" — كرّرها بنفس النبرة في كل جلسة |

### اسمع عشان تفهم الـ vibe:
- **Andy Puddicombe** (Headspace) — بطء + ابتسامة في الصوت
- **Tamara Levitt** (Calm) — دفء + حضور
- **Jeff Warren** (Ten Percent) — ذكي + مش دراما

**ملاحظة مهمة:** احنا مش عايزين accent of any meditation-teacher-voice. **عايزين صوتك الطبيعي لما بتحكي لصاحبك متضايق**. الناس بتعرف الفرق فوراً.

---

## 4. طريقة التسمية والتسليم

كل clip لازم يتسجل في **ملف MP3 منفصل**. تسميته مهمة جداً للكود يلاقيه:

### صيغة التسمية:
```
{sessionId}__{clipId}.mp3
```

مثلاً:
- `quick-calm__intro.mp3`
- `quick-calm__guide-1.mp3`
- `quick-calm__guide-2.mp3`
- `quick-calm__outro.mp3`
- `signature__welcome.mp3`  ← Signature عام لكل الجلسات

### مكان التسليم:
- اعمل Google Drive folder
- كل session في subfolder باسمها
- إبعتلي الـ link

### **مهم:** الملفات لما هتتضاف، الكود هيستخدمها تلقائياً. **مفيش حاجة تانية محتاج تعملها**.
