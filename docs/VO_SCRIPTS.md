# 🎙️ VO Recording Guide — واحة Meditation

**الهدف:** تسجيل 79 clip صوتي لـ 10 جلسات تأمل + signature + whispers.

---

## 📋 الإعدادات التقنية

| Parameter | Value |
|-----------|-------|
| Sample Rate | **48,000 Hz** |
| Bit Depth | **24-bit** for recording → **16-bit** for final |
| Format نهائي | **MP3 128 kbps stereo** |
| Loudness | **-16 LUFS** (Normalize) |
| Peak | **-1 dB** max |

### المكان + المعدات
- غرفة بسجاد أو دولاب هدوم مفتوح
- ميكروفون: **Shure MV7** أو **Rode NT1-A** (مش iPhone)
- مسافة من الميكروفون: **10-15 سم**
- Pop filter بين الفم والميكروفون
- **قفل التكييف والموبايل** قبل التسجيل

### الـ Performance
- **Pace بطيء** — أبطأ 30% من الطبيعي
- **صمت 2-3 ثوان** بعد كل جملة طويلة
- **Whisper zone** — جمل "هدّي / استرخي / سيب" همس خفيف
- **تنفّس على الميكروفون** بين الجمل — طبيعي ومش عيب
- **لا تبتسم** أثناء الـ whispers

# 🧠 لو الـ VO مش جاهز — الـ Fallback تلقائي

الكود ذكي:
1. يعمل HEAD request لـ `/meditation/vo/{id}.mp3`
2. لو الملف موجود → يشغّله
3. لو الملف مش موجود → يستخدم Web Speech API (صوت كمبيوتر) مع نفس النص

معناها: تقدر تسجّل الـ sessions على دفعات، والباقي يشتغل بـ TTS مؤقتاً. كل ما تضيف ملف جديد، الكود يستخدمه تلقائياً.

بس الـ TTS **مش ممتع** — هدفنا إن كل الـ 77 clip يبقوا بصوتك بأسرع وقت.
