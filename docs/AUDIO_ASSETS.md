# 🛠️ Tools للمعالجة

## Audio
| الأداة | الاستخدام | السعر |
|-------|----------|------|
| **Audacity** | Normalize + trim + format | مجاني |
| **Reaper** | Professional editing + EQ | $60 |
| **iZotope RX** | Noise removal (إزالة الكلّ البشع) | $299 |
| **Auphonic** | Auto-levelling + normalization (web-based) | مجاني حتى 2h/شهر |

## Video Compression
| الأداة | الاستخدام | السعر |
|-------|----------|------|
| **HandBrake** | Convert MP4 → WebM | مجاني |
| **ffmpeg** | CLI batch conversion | مجاني |
| **Shutter Encoder** | GUI بسيط | مجاني |

### ffmpeg command للـ videos
```bash
ffmpeg -i input.mp4 -c:v libvpx-vp9 -crf 35 -b:v 0 -an \
       -pix_fmt yuv420p -s 1920x1080 -r 30 output.webm
```

### Audacity macro للـ audio (Normalize → Export MP3)
```
Normalize → RMS -16 dB → Export as MP3 128 kbps
```
