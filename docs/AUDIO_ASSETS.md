# 🛠️ Tools

## Audio
- **Audacity** — free — للنورمالايز والتقطيع
- **Reaper** — $60 — للـ pro editing
- **Auphonic** — free up to 2h/mo — auto-level + normalize (web-based)

## Video
- **HandBrake** — free — MP4 → WebM conversion
- **ffmpeg** — free CLI:
```bash
ffmpeg -i input.mp4 -c:v libvpx-vp9 -crf 35 -b:v 0 -an \
       -pix_fmt yuv420p -s 1920x1080 -r 30 output.webm
```

# 🔄 Fallback System (موجود في الكود)

**أهم جزء:** الكود بيتعامل مع غياب الملفات بذكاء.

كل asset بيتفحص HEAD request:
```typescript
// هل الملف موجود؟
isAudioAvailable("/meditation/ambient/ambient__sea__loop.mp3")

// لو موجود → يشغّله
// لو مش موجود → يستخدم fallbackUrl (CDN preview من Freesound)
```

**معنى كده:** 
- تقدر تنشر الـ app **دلوقتي** والناس تسمع CDN preview
- كل ما تضيف ملف حقيقي، الـ app تلقائياً يشغّله بدل الـ fallback
- الـ cache في sessionStorage بيقلل HEAD requests
